# Incidentes

Severidad P0: exposición de secreto, pérdida de datos o autenticación rota.
P1: sync, publicación o login degradado. P2: funcionalidad no crítica.

Contén, conserva request IDs, rota secretos comprometidos, comunica impacto,
restaura el servicio y registra postmortem con causa, línea temporal y acciones.
