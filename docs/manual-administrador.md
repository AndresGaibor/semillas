# Manual de administrador

**Owner:** M6 · **Revisión:** 2026-07-13

El rol administrador habilita `/admin`. Desde allí se gestionan temas, pasos
CRECER, actividades, media, usuarios, reportes, revisiones y logros.

No publiques directamente desde un borrador: envía a revisión, registra notas y
usa la acción humana de publicar. Usa media privada, texto alternativo y evita
PII de menores. Cada cambio sensible queda en auditoría.
