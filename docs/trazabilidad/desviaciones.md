# Desviaciones aprobadas

## DESV-01 — PWA instalable en lugar de APK

Semillas se entrega como una única React PWA responsive instalable en Android,
iOS y escritorio. No se construye APK ni wrapper nativo separado. Esta decisión
mantiene una sola base de código y debe ser aprobada por Dirección antes del
cierre G8.

**Estado:** alcance aceptado por el solicitante; pendiente de firma formal de
Dirección en G8.
