# CMS editorial

**Owner:** M6 · **Revisión:** 2026-07-13

Flujo obligatorio: `borrador → revisión → publicado → archivado`.

Un tema completo debe tener senda, cita/version bíblica, franjas disponibles,
los seis momentos CRECER por franja, actividades válidas y media segura. La
vista previa permite revisar cada franja antes de enviar a revisión. El backend
revalida todo aunque se omita la interfaz.
