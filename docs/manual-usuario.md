# Manual de usuario

**Owner:** M2 · **Revisión:** 2026-07-13

1. Abre Semillas e instala la PWA desde el navegador.
2. Elige Google, Facebook o `Jugar como invitado`.
3. Selecciona una franja etaria y un apodo; no se solicita edad exacta.
4. Elige una senda, recorre CRECER y completa actividades.
5. Descarga el tema para jugar sin conexión.
6. Al volver internet, revisa el estado de sincronización.

Si aparece “requiere revisión”, conserva la aplicación abierta y reintenta con
internet. No compartas tokens ni datos personales.
